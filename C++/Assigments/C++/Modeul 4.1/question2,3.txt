Object Oriented Programming is a paradigm that provides many concepts such as inheritance, data binding, polymorphism etc. The programming paradigm where everything is represented as an object is known as truly object-oriented programming language.

3)difference between oop and pop.
OOP is a fundamental programming concept that developer uses for binding data and functions together using the concept of object and class, whereas POP it is a step-by-step approach to decomposing a task into a set of variables and routines using a set of instructions.
